# 一、功能介绍
- 普通用户
  - 注册、登录、找回密码
  - 发布二手房信息，管理自己的房源信息，上下架
  - 订单管理
  - 收藏管理
  - 在线私信

- 管理员功能
  - 全部房源管理，审核房源
  - 全部订单管理
  - 收藏管理
  - 用户管理
  - 新闻管理
  - 在线私信
  - 个人信息，修改密码
